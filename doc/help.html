PhpMathPublisher.Extended : help
PhpMathPublisher.Extended is improved PhpMathPublisher, represented by Pascal Brachet

How to use PhpMathPublisherExtended

1. Unpuck phpmathpublisher_v1.rar to your site root.
2. Require file 'phpmathpublisher/mathpublisher.php' for using this library in your script.
3. Use command: echo PHPmathpublisher_Units_Publisher::mathfilter ('{latex command}', {font_size}, {path_to_img}); to get html code, which represents nessary img tag.

For example:
<?php
require ('phpmathpublisher/mathpublisher.php');
echo PHPmathpublisher_Units_Publisher::mathfilter ('\root [4]{c}', 16, 'phpmathpublisher/Img/');
?>
